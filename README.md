# 摸鱼大作战 (Slack Master 2024)

> 像素风程序员职场生存模拟游戏

## 项目概述

本项目基于 Vue 3 + TypeScript + Pinia + UnoCSS 构建，实现了《摸鱼大作战》的核心原型。

### 已实现功能 (Phase 1 & 2)

- **核心架构**:
  - 基于 Pinia 的响应式状态管理 (Game, Player, UI Stores)
  - 模块化的游戏引擎 (GameEngine)
  - 类型安全的事件与行动系统

- **游戏机制**:
  - **核心循环**: 回合制行动系统 (每天 8 小时/回合)
  - **资源管理**: 精力(Energy)、摸鱼值(Chill)、进度(Progress)、怀疑度(Suspicion)
  - **行动系统**: 包含工作(Work)、摸鱼(Slack)、社交(Social)三大类行动
  - **事件系统**: 基础的事件触发机制 (如老板巡逻)

- **UI/UX**:
  - **像素终端风格**: 结合 Retro 与现代 AI 终端的视觉设计
  - **HUD**: 实时显示各项生存指标
  - **行动面板**: 可交互的行动选择区域
  - **事件日志**: 记录游戏过程的文字反馈
  - **响应式设计**: 适配桌面与移动端

## 快速开始

### 1. 安装依赖

```bash
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

### 3. 构建生产版本

```bash
pnpm build
```

## 玩法说明

1. **目标**: 在不被开除(怀疑度 < 80)和不累死(精力 > 0)的前提下，熬过 30 天。
2. **策略**:
   - **认真工作**: 降低怀疑度，增加进度，但消耗精力。
   - **摸鱼**: 恢复精力，增加快乐(Chill)，但增加怀疑度。
   - **社交**: 喝咖啡恢复精力，聊八卦获取信息。
3. **注意**: 每天 9:00 - 17:00 (8个回合)。注意老板巡逻事件！

## 技术栈

- **Framework**: Vue 3.5
- **Language**: TypeScript 5.7
- **Build**: Vite 8
- **Styling**: UnoCSS (Pixel preset)
- **State**: Pinia 2.2
- **Utils**: VueUse
